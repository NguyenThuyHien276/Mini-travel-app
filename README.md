# Mini-travel-app

KÍNH GỬI THẦY CÔ VỀ BÀI NỘP

MỘT SỐ LƯU Ý:
+ Backend (LLM Server): Được xây dựng bằng Ollama và đang chạy trên Google Colab.
+ Cổng kết nối: Frontend kết nối đến Backend thông qua đường tunnel Ngrok, có địa chỉ công khai là một Endpoint API được ghi trong file app.py.
